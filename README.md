**Index.html**  
html lang="id" digunakan untuk menunjukkan bahwa bahasa utama pada halaman web ini adalah Bahasa Indonesia. Bagian head berfungsi untuk menyimpan informasi dan pengaturan halaman web yang tidak ditampilkan langsung di layar browser. meta charset="UTF-8" berfungsi agar karakter, simbol, dan emoji dapat ditampilkan dengan benar oleh browser tanpa ada kesalahan pembacaan. meta name="viewport" digunakan agar tampilan website menjadi responsif dan otomatis menyesuaikan ukuran layar pada berbagai perangkat, mulai dari smartphone hingga laptop. title berfungsi untuk menentukan judul halaman "Simulasi Rangkaian Listrik" yang muncul pada tab browser. link rel="stylesheet" digunakan untuk menghubungkan file HTML dengan file CSS eksternal bernama style.css agar tampilan website terlihat lebih menarik. Bagian body berisi seluruh konten utama yang ditampilkan kepada pengguna. Tag h1 digunakan sebagai judul utama simulasi yang berada di bagian paling atas. div dengan class container berfungsi sebagai pembungkus utama kelompok lampu agar tata letaknya lebih rapi dan mudah diatur menggunakan CSS. Di dalam kontainer, digunakan tag img untuk menampilkan gambar lampu, di mana atribut id yang unik digunakan agar JavaScript dapat mengenali masing-masing lampu secara spesifik, dan atribut src menunjukkan status awal lampu (mati). Tag p memberikan keterangan untuk setiap jenis tombol kontrol. Tag button dengan atribut onclick berfungsi untuk menjalankan fungsi JavaScript tertentu saat tombol diklik oleh pengguna. Di bagian akhir, terdapat tag script yang berisi logika pemrograman. Fungsi toggleIndividu digunakan untuk membolak-balik status satu lampu secara mandiri, sedangkan fungsi toggleGrup menggunakan perintah forEach untuk menyalakan atau mematikan beberapa lampu sekaligus dalam satu waktu sesuai dengan daftar ID yang ditentukan. Penutup body dan html menandakan bahwa seluruh struktur halaman telah selesai.
**style.css**  
Kode style.css digunakan untuk mengatur tampilan visual dan tata letak seluruh elemen simulasi agar terlihat rapi dan interaktif. Pada bagian awal, body diatur agar teks berada di posisi tengah (text-align: center) dengan jenis font "Segoe UI" agar terlihat modern. background-image digunakan untuk memasang gambar latar belakang dari URL eksternal, yang dikombinasikan dengan background-size: cover agar gambar menutupi seluruh area layar secara proporsional. Properti background-attachment: fixed memastikan latar belakang tetap diam saat halaman di-scroll, sementara margin: 0 digunakan untuk menghilangkan jarak bawaan browser. Tag h1 diberi warna putih dengan text-shadow agar teks tetap menonjol dan mudah dibaca di atas latar belakang pemandangan yang berwarna. class container dirancang sebagai kotak putih transparan menggunakan rgba agar memberikan efek tembus pandang yang elegan, serta diberikan border-radius: 15px agar sudutnya melengkung dan box-shadow untuk memberikan efek bayangan yang dalam. Pada bagian img, ukuran lebar gambar diatur menjadi 80px agar seragam, dan diberi efek transition agar perubahan status lampu terlihat halus. Efek hover pada gambar menggunakan transform: scale(1.1) agar lampu sedikit membesar saat kursor diarahkan ke sana, memberikan kesan interaktif. Bagian button diatur dengan warna latar biru gelap, tanpa garis tepi (border: none), dan memiliki teks putih tebal. Efek hover pada tombol ditambahkan agar warna berubah menjadi lebih terang dan tombol sedikit bergeser ke atas (translateY) saat akan diklik. class title digunakan untuk memberikan gaya pada judul setiap rangkaian agar memiliki ukuran font yang lebih besar dan garis bawah sebagai pemisah konten yang jelas. Terakhir, tag p diatur dengan margin dan ketebalan font tertentu agar teks keterangan di atas tombol terlihat.
